## ⭐ Featured: SolarSurge (full-stack + cloud)

A dashboard + mobile app that pre-cools a home's AC when the solar yield is high,
instead of drawing grid power at the evening peak. My first end-to-end project
beyond the frontend: weather data in, a transparent cooling rule, and a live,
self-updating demo.

**[🌐 Live demo](https://solarsurge-web.vercel.app/)** · **[📄 Code & README](https://github.com/MarcelFelder-git/solarsurge)**

`Next.js` · `TypeScript` · `Python (AWS Lambda)` · `PostgreSQL` · `Prisma` · `Expo` · `AWS CDK` · `pytest`
